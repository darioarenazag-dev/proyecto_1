# ControlaTusCuentas

**ControlaTusCuentas** es una aplicación web ligera y funcional diseñada para ayudar a los usuarios a gestionar sus finanzas personales. Permite el registro de ingresos y gastos, ofreciendo una visión clara del flujo de dinero mensual.

---

## Características y Vistas

El proyecto consta de **3 vistas independientes** vinculadas mediante un sistema de navegación coherente:

1.  **Inicio (`index.html`)**: Es la página de bienvenida con acceso rápido a la función principal.
2.  **Registrar Transacción (`registro.html`)**: Es el formulario para capturar datos de ingresos o gastos (monto, categoría y fecha).
3.  **Resumen Mensual (`resumen.html`)**: Nos permite la visualización de las operaciones actuales mediante tarjetas de información.

### Instalación

1. Clona o descarga este repositorio.
2. Abre el archivo **(`index.html`)** en tu navegador preferido. No requiere servidores ni dependencias externas.
